# Monitor presence

## As super admin

You can subscribe to 

```
/restapi/v1.0/account/~/extension/${ext.id}/presence?detailedTelephonyState=true
```

And monitor other's presence.


## As standard user

You can subscribe to 

```
/restapi/v1.0/account/~/extension/${ext.id}/presence?detailedTelephonyState=true
```

And monitor other's presence.
